### 객체 배열
다양한 데이터를 그룹화하고 효율적으로 관리하기 위해서 배열 안에 객체를 만들어 사용하는 것을 `객체 배열`이라 부른다.
```js
const students = [
    {
        name: "a",
        age: 12,
        blood-type: 'a'
    },
    {
        name: "b",
        age: 12,
        blood-type: 'o'
    },
    {
        name: "c",
        age: 14,
        blood-type: "ab"
    }
]

module.exports = students
```
